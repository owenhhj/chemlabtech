## Documentation Site for Chemistry Instructional Technology

### Hosting

Hosted on cPanel.

### Repository Structure

Home page `index.html` contains links to subject-specific pages for audio, video, internet, etc.

`media/` stores all media content to be seen in the documentation, including pictures, videos, etc.

`scripts/jquery-version.min.js` is a JavaScript library empowering the functionalities of this website.

`issues.json` is the database storing all the troubleshooting Q&As of this website.

`scripts/renderer.js` reads the issues in `issues.json` and renders them into the website.
